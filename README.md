# biocrystal

BioCrystal is an open source resource aiding the development of bioinformatics software.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     biocrystal:
       github: leandroradusky/biocrystal
   ```

2. Run `shards install`
